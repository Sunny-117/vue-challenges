# Win11 in Vue! 

## 你从未体验过的全新版本！

该项目作为第二届阿里巴巴练习生计划“重生之我是尤Vue溪”小组的开发项目，旨在使用 HTML、CSS (SCSS) 和 JS 等 Web 技术和 Vue 框架，在 Web 上复刻 Windows 11 的 UI 体验。

**本项目只做交流学习使用，禁止用于其他用途！**

点击[http://101.34.216.21/#/open](http://101.34.216.21/#/open)进入在线体验网站。
